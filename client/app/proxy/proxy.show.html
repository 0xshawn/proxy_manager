Step 1: Install APK

  • Goto Shadowsocks Android Releases page and download the latest version of APK.
  • Install the downloded apk into your Android.

Step 2: Configure Shadowsocks

  • Open Shadowsocks and configure with proxy setting.

  • Check UDP Forwarding

Step 3: Enable Proxy

Touch the Toggle Button at the top right corner and enjoy.

Step 1: Install Shadowsocks C# Client

  • Download a latest release.
    For >= Windows 8 or with .Net 4.0, download Shadowsocks-win-dotnet4.0-x.x.x.zip.
    For >=W indows 7 or with .Net 2.0, download Shadowsocks-win-x.x.x.zip.

Step 2: Usage

  • Find Shadowsocks icon in the notification tray
  • You can add multiple servers in servers menu
  • Select Enable System Proxy menu to enable system proxy. Please disable other proxy addons in your browser, or set them to use system proxy
  • You can also configure your browser proxy manually if you don't want to enable system proxy. Set Socks5 or HTTP proxy to 127.0.0.1:1080. You can change this port in Server -> Edit Servers
  • You can change PAC rules by editing the PAC file. When you save the PAC file with any editor, Shadowsocks will notify browsers about the change automatically
  • You can also update the PAC file from GFWList. Note your modifications to the PAC file will be lost. However you can put your rules in the user rule file for GFWList. Don't forget to update from GFWList again after you've edited the user rule
  • For UDP, you need to use SocksCap or ProxyCap to force programs you want to proxy to tunnel over Shadowsocks
